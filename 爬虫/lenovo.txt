报告效率提升 95%电子商务策略更加有的放矢人力资源部门的 100 种报告整合成战略仪表板联想是一家全球性科技公司，在全球拥有超过 55,000 名员工，客户遍及 160 多个国家/地区。借助 Tableau，联想公司的分析、商业智能和可视化团队制作了一个灵活的销售仪表板，各个部门都可以调整这个仪表板来进行临时分析。得益于这一点，该公司在 28 个国家/地区的效率提升了 95%。电子商务团队通过分析客户互动指标，设计出了更出色的在线体验，这不仅提升了品牌认知度，还创造了更丰厚的收入。人力资源部门则将 100 多个静态报表整合成了一组战略仪表板，以此鼓励采用数据驱动型方法来达到团队的各个里程碑。如今该公司在 15 个业务部门有大约 3,000 名活跃的 Tableau 用户，公司内访问 Tableau 仪表板的用户数超过 10,000 名；对此，分析、商业智能和可视化团队负责人 Santhosh Nair 向我们解释了“实时决策机制如何帮助相关负责人员更快地制定出业务决策”。我们实际上可以用节省下来的这些宝贵时间深入分析业务情况，将分析洞见反馈给决策者，从而帮助他们做出决策并采取行动，最终让公司受益。

          Santosh Nair, 
分析、商业智能和可视化团队负责人
以前，联想公司是以 Excel 格式制作一份统一的销售报告，然后传送到 28 个不同的国家/地区。当各部门和区域想要调整该报告时，就需要八到十个人参与进来，导致分析团队的任务量不堪重负。分析、商业智能和可视化团队负责人 Santhosh Nair 采用 Tableau 作为联想的分析解决方案后，整个团队的效率得到了极大的提升。 Santhosh 透露：“过去，我们可能需要六七个小时才能制作好一份周报告。  每份报告花 7 小时，不难想象 30 份报告要花多长时间。使用 Tableau 后，我们的效率提升了 95%。大家得以将更多的时间花在完成各自真正的工作上，而不是耗费在制作报告上。”大数据架构高级经理 Marc Gallman 则表示：“我们建立好有效运行的数据处理做法后，立即就可以每天甚至每小时提供一次数据。”联想收获的改进并不仅仅体现在 95% 的效率提升方面。Marc 打算不断提升数据端的性能：“我们要通过 Tableau 实现的目标是，对于仪表板中的任何一次单击，都要确保响应时间不超过 5 秒。”使用 Tableau 后，我们的效率提升了 95%。大家得以将更多的时间花在完成各自真正的工作上，而不是耗费在制作报告上。


          Santosh Nair, 
分析、商业智能和可视化团队负责人
Santhosh 表示：“我们实际上可以用节省下来的这些宝贵时间深入分析业务情况，将分析洞见反馈给决策者，从而帮助他们做出决策并采取行动，最终让公司受益。”该团队可以轻松地从 Amazon Web Services 和 Hortonworks Hadoop Hive 等来源提取数据。“这些引擎的运行速度越来越快，因此我们可以更快、更高效地提供仪表板或分析洞见。”借助 Tableau，各部门如今在获取综合报告后，可以根据自己的需求进行自定义，这样一来，临时分析请求得以减少，他们也获得了更加贴合自己需求的仪表板解决方案。该报告现在每天更新三次之多，不再是一周只更新一次，从而确保了数据处于最新状态。Santhosh 透露：“我们改用了统一的整合报告，报告中提供了很多交互功能。大家可以自行查看这些仪表板，提出不同的业务问题，提前获得所需的答案，进而更快地做出决策。”随着高管们日渐了解可视化分析的价值，Tableau 在整个联想公司内的用户采用率持续攀升。联想起初部署的是 8 核 Tableau Server 实例，但很快就扩展到 16 核，以满足用户增长的需要。如今，公司内已有超过 10,000 名用户访问 Tableau 仪表板。 全球商业智能总监 Ashish Braganza 透露：“起初我们只是为了满足一种范围非常有限的业务使用情形，结果一发而不可收，它快速传播开来。Tableau 让大数据分析变得轻松简单，虽然人人都在谈论这方面的话题，但 Tableau 做出了行动，将它化繁为简。”Ashish 则表示：“我们最开始使用 Tableau 时，考虑的只是仪表板和报告。我们从未想到 Tableau 会让整个组织脱胎换骨。”Ashish 解释了随着采用范围的不断扩大，联想的公司文化如何转而采用更加以数据为中心的方法来化解挑战。 “它所带来的不仅仅是一种解决方案或技术，而是改变了在数据方面的文化。”为了在这种协作过程中有效管控联想的数据，该团队将单点登录功能与 Tableau Server 集成在了一起，以便为员工提供更好的体验。Tableau Server 根据本地身份存储区中存储的站点角色来管理用户对 Tableau 资源的访问。这有助于联想大规模地支持自助式分析。客户体验分析团队中负责测试与个性化工作的高级经理 Josh Fitzhugh 解释道：“得益于 Tableau 提供的便利性和帮助，我们团队才有办法分析所有这些海量的总体数据，设计出一种适合所有人的大规模交付机制。” Ashish 说：“我是永远都持怀疑态度的人，而 Tableau 以迅雷不及掩耳之势在整个组织内普及起来，着实让我大吃一惊。”我们最开始使用 Tableau 时，考虑的只是仪表板和报告。我们从未想到 Tableau 会让整个组织脱胎换骨。它所带来的不仅仅是一种解决方案或技术，而是改变了在数据方面的文化。

全球商业智能总监
在竞争激烈的电子商务领域，联想需要保持敏捷性。Santosh 表示：“如果我们不能适应，就会落于人后，也无法尽快做出改观。”为了支持这种快节奏的发展，联想的分析团队从他们的统一客户智能平台 (LUCI Sky) 将数据提取到 Tableau 中，以分析他们电子商务网站上的客户体验。Tableau 仪表板会显示客户在其行为历程的每个阶段是如何与内容交互的，从而帮助联想团队更加有效地锁定目标客户，并最终售出更多产品。Santosh 解释道：“Tableau 帮助我们绘制出了各个阶段的互动历程，从浏览产品到购买后的情形，都一目了然。包括最高管理层在内的整个组织都在根据这些历程做出日常决策。”在近期刊登在 Analytics India 杂志上的一篇文章中，Santosh 分享了 Tableau 如何帮助联想团队整合多个数据源，从而全面了解客户体验。 Santosh 透露：“我们通过整合 30 个不同的数据源，制作了一个独特的仪表板。所有这些数据源都包含有关客户反馈的数据，反馈可能来自社交媒体、零售商网站或调查问卷。我们分析了所有这些数据并进行了自然语言处理，以了解其中反映的主题或客户对联想产品的看法。”“实时决策机制正在帮助相关人员更快地制定出业务决策，因此销售额和收入等关键成功指标自然也就会节节攀升。”分析团队让联想公司的多个部门如虎添翼。人力资源部门就是其中一个。 在使用 Tableau 前，人力资源团队在全球范围内需要制作大约 100 多种报告，涉及人才招聘、薪酬、销售佣金、人员流失和人员编制管理等诸多方面。这导致报告周期十分漫长，使得人力资源部门的领导者难以做出数据驱动型决策。使用 Tableau 刚刚三个月，联想的人力资源部门就制作出了一系列以数据分析为依据的战略仪表板，从而帮助他们达成了该部门的一个个重大里程碑。
